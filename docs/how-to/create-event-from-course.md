Use this workflow when you already have a course structure and want to deliver it to a cohort as a scheduled event.

## Before you start

You need admin access and an existing course that represents the teaching structure you want to reuse.

## Steps

1. Open the event creation flow from the home page or events area.
2. Choose **From course** instead of **Blank event**.
3. Set the event start date and time.
4. Select the source course from the course picker.
5. Create the event.
6. Open the new event's edit tab and adjust any event-specific details:
   - title and summary
   - dates, locations, and group timings
   - enrolment and instructor keys
   - any material changes needed for this cohort

## What happens next

The new event starts with copied groups and material from the course, but can then diverge freely as its own scheduled delivery.
