Use this workflow when you want a reusable learning path that people can enrol on and work through in their own time.

## Before you start

You need admin access and material already available in Gutenberg.

## Steps

1. Go to the courses area and choose **Add course**.
2. Create the course manually or import grouped course JSON.
3. Open the new course page and switch to edit mode.
4. Add one or more material groups.
5. Add material sections inside those groups and reorder them into the teaching sequence you want.
6. Fill in the course summary, level, languages, tags, and learning outcomes.
7. Save the course and review the public course page.

## What happens next

Learners can enrol on the course, select it as their active course, and progress through the grouped material with course-aware hints and navigation.
