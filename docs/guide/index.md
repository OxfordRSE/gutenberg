## Courses, events, and material pages

Gutenberg is no longer just an event-management wrapper around material pages:

- **Courses** provide reusable, grouped, self-paced learning paths.
- **Events** provide dated teaching deliveries, optionally created from a course blueprint.
- **Material pages** can now reflect the learner's active course or event and show context-aware hints and navigation.

If you are new to the platform, read the Courses and Events pages first, then the Learning Context page.
