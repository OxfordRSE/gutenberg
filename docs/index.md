# Gutenberg

Gutenberg is a teaching site for structured markdown-based learning material. It supports two main ways of using that material:

- **Self-paced courses** built from curated groups of material sections.
- **Scheduled events** for live teaching, workshops, and cohort delivery.

It includes active course and event context, progress tracking, grouped course editing, and event creation from course blueprints.

## Courses and Events

Use **courses** when you want a reusable, self-paced learning path with grouped material and tracked progress. Use **events** when you want to deliver the material to a cohort with dates, locations, enrolment, instructors, and event-specific structure.

Material still lives in one or more configured markdown repositories. Gutenberg adds the teaching layer on top: default courses, course sync, active course and event context, grouped navigation, and admin workflows.
